3 #mixins Mixins

This is mixins you can use

3.1 #mixins.rem rem($property, $values)

  • $property
    CSS property, must be surrounded by quotes "font-size".
  • $values
    CSS appropriate size value in pixels or rem, 10px or 1rem.

Outputs responsive unit sizing for given property and size value.
